# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Ayush-Coder077/portfolio.git
```

Go inside

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
